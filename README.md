# TubeDownloader
You can download movie/audio from Youtube with python.

# Requirement
pyTube3 9.6.4
moviepy 1.0.3

# Installation

```bash
pip install pyTube3
pip install moviepy
```
# Usage
1.git clone https://github.com/ZenkiKawakami/TubeDownloader.git
2.cd TubeDownloader
3.python TubeDownloader.py
